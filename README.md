# 🤖 AI Code Reviewer

An AI-powered code review application that analyzes source code using Google's Gemini AI and provides detailed feedback, suggestions, and improvements in a clean and interactive interface.

🌐 **Live Demo**
- Frontend: https://ai-code-review-frontend-776i.onrender.com
- Backend API: https://ai-code-review-m4xu.onrender.com

---

## 📌 Features

- ✨ AI-powered code review using Google Gemini
- 📝 Monaco Code Editor (VS Code-like editor)
- 🎨 Beautiful Markdown rendering
- 💻 Syntax highlighted code blocks
- ⚡ Fast Node.js + Express backend
- ⏳ Loading animation while AI generates response
- 📖 Detailed review including:
  - Overall Review
  - Strengths
  - Problems
  - Suggested Improvements
  - Improved Code
  - Complexity Analysis
  - Final Rating
- 🌐 Fully deployed on Render

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/chaitanya1013/AI-Code-Review.git
```

```bash
cd AI-Code-Review
```

---

# Backend Setup

Navigate to backend

```bash
cd BackEnd
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Start backend

```bash
node server.js
```

Server runs on

```
http://localhost:3000
```

---

# Frontend Setup

Navigate to frontend

```bash
cd Frontend
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
VITE_API_URL=http://localhost:3000
```

Run frontend

```bash
npm run dev
```

Open

```
http://localhost:5173
```

## 🚀 API Endpoint

### Review Code

**POST**

```
/ai/get-review
```

### Request

```json
{
  "code": "function sum(){ return 1+1; }"
}
```

### Response

Markdown generated by Gemini AI

```markdown
# Overall Review

...

# Strengths

...

# Problems

...

# Suggested Improvements

...
```
